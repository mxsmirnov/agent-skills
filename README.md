# agent-skills
Навыки ИИ-агентов

Данный репозиторий содержит набор **agent skills** (навыков) — модулей, которые расширяют возможности ИИ-агентов, предоставляя им специализированные инструкции, скрипты и ресурсы для выполнения конкретных задач.

## Что такое Agent Skill?

**Agent Skill** — это самодостаточный модуль, который наделяет агента знаниями и инструментами для выполнения определённого вида деятельности. Навык включает:
- описание задачи и инструкции для агента;
- скрипты и утилиты;
- справочные материалы;
- дополнительные ресурсы (изображения, конфиги и т.д.).

Skills могут быть подключены к агенту динамически, позволяя наращивать его функциональность без изменения основного кода.

## Стандарт Agent Skills

Репозиторий следует открытому формату **Agent Skills** — простому способу дать агентам новые возможности и экспертизу.  
Skills представляют собой папки с инструкциями, скриптами и ресурсами, которые агенты могут обнаруживать и использовать для более точного и эффективного выполнения задач.

Подробнее: [Agent Skills](https://agentskills.io)

## Типовая структура навыка

Согласно стандарту, навык организуется в директории со следующей структурой:
``` 
skill-name/
├── SKILL.md    # Основной файл с описанием навыка и инструкциями для агента
├── scripts/    # Исполняемые скрипты (Python, bash и т.п.)'
├── references/ # Документация, схемы, дополнительные справочные данные
└── assets/     # Изображения, файлы конфигураций, прочие статичные ресурсы
```

## Добавление новых навыков

Чтобы добавить новый навык:
1. Создайте папку с именем навыка в корне репозитория.
2. Разместите в ней файл `SKILL.md` с обязательным описанием и инструкциями.
3. Организуйте вспомогательные файлы по папкам `scripts/`, `references/`, `assets/` при необходимости.
4. Обновите данный README, добавив краткое описание нового навыка в раздел «Навыки в репозитории».
